# ENDPOINT

-   [POST] /api/auth/sign-up
    -   회원 가입
-   [POST] /api/auth/sign-in

    -   로그인

-   [GET] /api/event

    -   모든 이벤트 조회

-   [GET] /api/event/:eventid

    -   특정 이벤트 조회

-   [POST] /api/event

    -   이벤트 생성

-   [GET] /api/event/:eventid/reward

    -   이벤트 보상 정보 조회

-   [POST] /api/event/reward
    -   이벤트 보상 정보 생성

# 실행 방법

-   docker-compose up --build
